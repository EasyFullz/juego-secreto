Funciones
1

Monitoreo y alerta

El sistema cuenta con una tecnología que nos permite crear un perímetro virtual, este mismo detecta tanto si se entra o sale de él.

Cuando este perímetro es rebasado, se activan automáticamente las sirenas y luces estroboscópicas durante 10s siempre y cuando no se detecte mas movimiento en el perímetro.

2

Cerca electrica

El sistema cuenta con una tecnología que nos permite crear un perímetro virtual, este mismo detecta tanto si se entra o sale de él.

3

Activación manual

El sistema de seguridad también puede ser activado por medio de un pulsador de pánico desde la oficina en la zona de cartera.

4

Desactivacón del sistema

En caso de que el sistema presente fallos y falsos positivos se puede desactivar mediante el pulsador de emergencia correspondiente.

El pulsador con caja naranja desactiva las alarmas que sean accionadas por el paso del perímetro virtual.

El pulsador con caja amarilla desactiva las alarmas accionadas por la cerca eléctrica.

Recomendaciones
  • 1

    Se recomiendo hacer un mantenimiento periódico de todo el sistema de seguridad tanto hardware como software para tener un correcto funcionamiento de este.

  • 2

    Mantener el software del sistema actualizado para garantizar la seguridad y eficiencia del sistema.

  • 3

    Hacer pruebas de funcionamiento periodicamente para confirmar que el sistema funciona correctamente

  • 4

    En caso de tener problemas del sistema mas alla de sus conocimiento contactarse con soporte.

  • 5

    En caso de tener problemas de daño de alguno de los componentes del sistema de seguridad, cambiarlo lo mas pronto posible
